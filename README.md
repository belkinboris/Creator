# Создатель v0.1

От идеи до денег по этапам, которые нельзя перепрыгнуть.
Сейчас работают этапы ⓪ (Оффер) и ① (Спрос): идея → 3 заострённых
оффера → сгенерированный и захощенный smoke-лендинг → сбор событий →
детерминированный вердикт по спросу.

## Деплой (Railway, отдельный проект — НЕ смешивать с Аналитиком)
1. Новый репозиторий на GitHub из этой папки.
2. Railway → New Project → Deploy from GitHub.
3. Variables: ANTHROPIC_API_KEY (отдельный ключ "sozdatel"),
   DATABASE_URL (Railway Postgres или оставить sqlite для старта).
4. Start command: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`

## Проверка после деплоя
— `/health` → {"ok": true}
— Открыть `/`, ввести идею → 3 оффера → «Запустить» → открыть `/l/{idea_id}`
— На лендинге отправить тестовую заявку → `/api/verdict/{idea_id}` покажет её.

инъекция LLM, markdown-обёртки), генерация лендинга (все слоты заполнены),
